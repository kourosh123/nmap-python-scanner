# 🔐 Nmap Python Scanner

## 📌 Overview
A simple Python-based network scanner using Nmap for learning cybersecurity and penetration testing basics.

---

## ⚙️ Features
- User input target (IP/domain)
- Runs Nmap scan automatically
- Displays open ports and services
- Saves output to a text file

---

## 🛠️ Technologies
- Python 3
- Nmap

---

## 🚀 How to Run

```bash
python3 scanner.py
